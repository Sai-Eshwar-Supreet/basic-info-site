# basic-info-site

A simple Node.js server that serves static HTML pages based on the request URL.

## Routes

Core capabilities:

- / or /home → Home
- /about → About
- /contact → Contact
- Any other route → 404

## Run Locally

### Setup

```bash
node index.cjs
```

Open: http://localhost:8080
